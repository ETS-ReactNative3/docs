## User scenario

Let's assume that user want to chat with touching chat icon button at other person's profile screen.
If init function is completed with success each other, both of them are ready to join chat.
Then like as below, navigate to following screen with peer member id.

```javascript
// Chat.js file that you generated

import Chatty from 'react-native-chatty-cloud';

interface ChatProps {
  at: string,
  with: string,
}

const Chat = (props: ChatProps) => {
  const user = useUser();
  return (
    <Chatty
      at={props.route.params.at}
      with={props.route.params.with}
      distinctKey=''
      group=''

      headerBg={'white'}
      headerLeftOnPress={() => props.navigation.goBack()}

      footerBg={'white'}
      rightBubbleColor={'green'}
    // TextInput Area
    // textInputPlaceholder='Enter Message'
    // attachIcon={<AddIcon />}
    // sendIcon={<ArrowForwardIcon />}
    />

  );
};

export default Chat;


```
Chatty가 init이 완료되면 member의 id와 이름이 chatty cloud에 등록이되며 채팅멤버로서 chat에 참여할수 있는 조건이 됩니다.
채팅방으로 들어가는 방법은 아래와 같습니다. 
- 상대의 member id를 가지고 둘만의 private chat 을 만드는방법
- chat list에서 하나의 chat을 선택해서 chat으로 진입
- createChat 함수를 이용하여 chat 생성
